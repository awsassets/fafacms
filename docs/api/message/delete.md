# /message/delete

将收到的站内信删除。

## 请求

```
POST /message/delete

{
	"ids": [
		45,
		46,
		24
	]
}
```

参数说明：

| 字段   |      含义   | 类型  |   参数 |  必选 |
|----------|--------|------|------|------|
| ids | 信息的ID列表 | array |  | 是 |

收到方的删除。

删除为逻辑删除，管理员列出接口仍然可以查看。

## 响应

正常：

```
{
  "flag": true,
  "cid": "c1a7576abafc4d07a168553a6d90938a"
}
```
