# /message/read

将未读的站内信设置为已读。

## 请求

```
POST /message/read

{
	"ids": [
		45,
		46,
		24
	]
}
```

参数说明：

| 字段   |      含义   | 类型  |   参数 |  必选 |
|----------|--------|------|------|------|
| ids | 信息的ID列表 | array |  | 是 |

## 响应

正常：

```
{
  "flag": true,
  "cid": "c1a7576abafc4d07a168553a6d90938a"
}
```
