
# /message/private/delete

删除私信。删除后自己看不到，但是收件人仍然可以看到。

## 请求

```
POST /message/private/delete

{
	"ids": [
		43
	]
}
```

参数说明：

| 字段   |      含义   | 类型  |   参数 |  必选 |
|----------|--------|------|------|------|
| ids | 信息的ID列表 | array |  | 是 |

接收方的删除。

删除为逻辑删除，管理员列出接口仍然可以查看。


## 响应

正常：

```
{
  "flag": true,
  "cid": "cf20e1be6de445248c1a92501955c91c"
}
```
